# Tunga App

Restaurant application with Spring Boot backend and static frontend: table reservations, delivery orders, and menu browsing.

## Structure

- **backend/** — Spring Boot API (Java, Maven)
- **frontend/** — HTML/CSS/JS (see [frontend/README.md](frontend/README.md))

## Quick start

- **Backend:** `cd backend && mvn spring-boot:run`
- **Frontend:** Open `frontend/index.html` in a browser or serve the `frontend` folder with any static server.

## Pushing to GitHub

1. Create a new repository on [GitHub](https://github.com/new) (do **not** initialize with README).
2. In this folder, run:

   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git branch -M main
   git push -u origin main
   ```

   Replace `YOUR_USERNAME` and `YOUR_REPO_NAME` with your GitHub username and repository name.
